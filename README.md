# BJVA 
Voilier miniature Autonome

based on BeagleBoneBlue

required
    librobotcontrol


List of waypoints: waypointlist.dat
    Should contains at least 2 waypoints
    One line per waypoint
    Latitude and Longitude in decimal, separated by a space
    Exemple:
        46.683924 -71.878586
        46.685768 -71.838973
        46.668241 -71.791758
